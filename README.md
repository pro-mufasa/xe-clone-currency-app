# 🌍 Mufasa Converter - Rwanda Exchange Web System
Mufasa Converter is a web application built using HTML, CSS, and JavaScript. It is designed to provide users with a fast and reliable way to convert various currencies, with a specific focus on the Rwanda Franc (RWF).

## Features

* Multi-language Support:** The platform supports three languages: Kinyarwanda, English, and French.
* Currency Conversion:** Real-time conversion between major currencies including USD, EUR, GBP, and RWF.
* Dark Mode Support:** A toggleable visual theme that adjusts based on user preference (Light/Dark mode).
* Conversion History:** Automatically saves your recent conversion activities using `localStorage`.
* Responsive Design:** Fully optimized for all devices, including desktops, tablets, and smartphones.

 Tech Stack

* **HTML5:** For semantic web structure.
* **CSS3:** For styling, layout design, and smooth UI animations.
* **JavaScript (Vanilla):** For mathematical logic, language switching, and data persistence.

 File Structure

* Home.html - The main landing page and primary converter interface.
* Full Converter.html - Extended conversion features and tools.
* Currency Details.html - Information and specifications for supported currencies.
* History.html - Displays the log of previously performed conversions.
* Setting.html - User configuration for language and theme preferences.
